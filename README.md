# PhotoMakerApp
Photomaking application for filter application, brightness correction and crop images.

## Filters
* Blur
* Sharpen
* Edge detection
* Grayscale
* Binary
* Negative
* Cartoon

## Value correction
* Hue
* Saturation
* Brightness

## Crop tool
* Crop
* Mirror effect
* Flip effect

## Selection tool
Lazo tool to select a sub-image for the application of the filters.

## Format support
* BMP
* JPG
* JPEG
* PNG
* PPM
